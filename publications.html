Instance Segmentation of Peripheral Blood Smear and Refining Classification via Domain Adaptation
Jimut Bahan Pal, Aniket Bhattacharyea, Debasis Banerjee and Tamal Maharaj
Under Process in Springer Nature Computer Science Journal

TL;DR

We have created a novel peripheral blood smear dataset for multiclass instance segmentation. We performed Domain Adaptation to increase the metrics with the help of another similar dataset. The method can be further used to automate diagnosing diseases via blood test and also annotate newer blood smear slides with minimum human intervention.
Improving Multi Scale Attention Networks - Bayesian Optimization for Segmenting medical images
Jimut Bahan Pal and Dripta Mj
Under Process in Computer Methods and Programs in Biomedicine Update Journal (Elsevier)

TL;DR

We have successfully created a computationally efficient model to segment medical data with the help of attention networks. We apply Bayesian Optimization to increase the segmentation capability of the existing network to achieve SOTA. We also test different attention modules and their effect on segmenting different datasets.
Holistic network for quantifying uncertainties in medical images
Jimut Bahan Pal
Accepted at BrainLes 2021 MICCAI Workshop, to be published in Springer LNCS

TL;DR

We have created Holistic Network, which is very efficient in segmenting uncertainty in medical image with less data. This overcomes the complication of using multiple U-Nets to perform the same task, simultaneously surpassing previous SOTA results in a few tasks.
