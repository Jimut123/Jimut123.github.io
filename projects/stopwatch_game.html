Purpose :


This game was developed by me during my association with the MOOCs in Principles of Computing course. This is a stopwatch game where you need to stop the stopwatch in whole number of time. This is developed in Python2 in codeSkulptor platform. The Github link and Youtube link are provided for demonstration. You can also see the link to the codeSkulptor platform, where you can see the game in live execution, just hit the play button to play!
