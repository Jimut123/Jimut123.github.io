ULTIMATE DATING PLACE FINDER

This project deals with the finding of ultimate place to date according to the preferences. Now you can find the best place to date for any country. The idea is to use the Foursquare API to collect information about the neighbourhood of the place that is provided. So we will collect data, and make this a geo-scatter-plot. We will use k-means to cluster that to some extent to get the best place which have maximum of those preferences. The preferences are given by the user and a country/ location is selected! then we will able to visualise the best place to date. The github link is provided for demonstration.
